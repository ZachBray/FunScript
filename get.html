Get FunScript today

Using NuGet

FunScript is available on NuGet. To reference FunScript library in Visual Studio project, just run the following command in the Package Manager Console:

PM> Install-Package FunScript

Direct download

Alternatively, you can also download the FunScript libraries in a single ZIP file. Before extracting the file, make sure to Unblock it to avoid loading errors.

Download ZIP file

Build from source

If you want to get the latest version or if you're considering contributing to FunScript, then you can get the source code from GitHub. For more information, see also our contributor guide!
